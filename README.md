# RadarFut
**RadarFut** é um projeto de visão computacional voltado para análise visual de partidas de futebol. A proposta é capturar a tela de jogos ou vídeos em tempo real, identificar jogadores em campo, diferenciar equipes por cores e representar suas posições em um minimapa tático semelhante ao radar utilizado em jogos como EA FC.  Inicialmente, o projeto será desenvolvido como uma aplicação desktop em Python, com foco em validar a viabilidade técnica da detecção, rastreamento e renderização dos jogadores em um campo 2D. Futuramente, a solução poderá evoluir para uma extensão de navegador ou overlay integrado, permitindo análises visuais mais práticas durante transmissões, replays ou partidas digitais.
